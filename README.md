# Python EDA Projects 📊

A collection of Exploratory Data Analysis (EDA) projects built using Python.  
This repository focuses on data cleaning, visualization, statistical analysis, and extracting meaningful insights from real-world datasets.

The purpose of these projects is to strengthen practical data analysis skills and build a strong foundation for Machine Learning and Data Science workflows.

---

# Projects Included

## 1. Titanic Dataset EDA

Performed Exploratory Data Analysis on the Titanic dataset to understand passenger survival patterns based on features such as gender, age, passenger class, and fare.

### Key Analysis
- Survival rate by gender
- Passenger class analysis
- Age distribution
- Missing value handling
- Correlation analysis
- Data visualization

### Future ML Scope
- Survival prediction model
- Classification using Machine Learning algorithms

---

## 2. Netflix Dataset EDA

Performed Exploratory Data Analysis on the Netflix Movies and TV Shows dataset to identify trends in content distribution, genres, ratings, and release patterns.

### Key Analysis
- Movies vs TV Shows distribution
- Country-wise content analysis
- Genre trends
- Ratings analysis
- Year-wise release trends
- Missing value treatment

### Future ML Scope
- Recommendation systems
- Trend prediction
- Content popularity analysis

# Installation & Usage

Clone the repository:

```bash
git clone https://github.com/anshere0/python-eda-projects.git
```

Move into the directory:

```bash
cd python-eda-projects
```

Install dependencies:

```bash
pip install pandas numpy matplotlib seaborn notebook
```

Run Jupyter Notebook:

```bash
jupyter notebook
```
